PhoDF - Photo PDF

  • Download FREE from Apple iOS app store

This app creates PDFs from photos, optimizes photos, and shows the file size of the PDFs while you create them, so that you won't hit PDF file size limit.

When you submit PDFs to your government or bank (for marriage proof, loan etc.), they usually have a PDF file size limit. You cannot submit PDFs big than 5MB (for example).

This app allows you to create PDFs from photos:

  • It optimizes the photos for you, so that you won't hit PDF file size limit.
  • While you create a PDF, this app shows the current PDF file size to you, so that you know how big the PDF is currently is.
